### 使用方式：
**直接打开**下载的工具即可。

### 解锁方式：
同时按下 Windows 徽标键和 R，在弹出的“运行”窗口处输入 cmd 并回车，进入 CMD，粘贴并运行以下命令（粘贴完成，记得回车）：

```
runas /user:administrator cmd
sc stop TDFileFilter 
sc delete TDFileFilter 
netsh winsock reset
```

即可解锁 U 盘和网络上的资料。
